# thingsboard-widget

## 聊天室部件：chatroom.json
用法：仪表板数据源添加任意一设备，遥测名称为“chat”。时间窗口数据聚合功能选“空”，一定！

## 白板部件：drawing-board.json
用法：仪表板数据源添加任意一设备，遥测名称为“draw_event”。时间窗口数据聚合功能选“空”，一定！
